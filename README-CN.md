## ❗ 常见问题排解

### n8n 无法启动

* 检查 Docker 是否正在运行: `docker ps`
* 检查端口冲突: `lsof -i :5678`
* 查看错误日志: `docker-compose logs n8n`

### Webhook 无法接收请求

* 确认 WEBHOOK_URL 环境变量设置正确
* 使用 ngrok 进行本地测试
* 检查防火墙设置

### API 认证失败

* 确认 API Key 有效且有额度
* 检查环境变量是否正确加载
* 验证 API endpoint 是否正确

### 工作流程执行错误

* 查看 n8n 执行日志
* 使用 Error Workflow（参考 workflow #18）
* 检查节点配置是否正确
