## 🎯 目標

學會用 n8n 結合 AI，自動標籤分類、摘要、回復郵件，優化 Gmail 工作流程。

* 1️⃣ **自動標籤 - 狀態**：處理輕重緩急
* 2️⃣ **自動標籤 - 用途**：根據場景回想
* 3️⃣ **重要郵件 - 總結報告**：快速瀏覽今天重要郵件
* 4️⃣ **911分類 - 自動回覆草稿**：回覆郵件從 60 分開始

## 🛠️ 對應情境

* 每天收信量大，郵件內容繁雜，花至少一個小時再整理。
* 想自動分類/刪除/歸檔郵件，減少重複瑣碎工作來增加生產力。
* 每日郵件自動劃重點，自動摘要更快掌握工作重心。
* 希望用 workflow 工具快速整理收件匣內容
