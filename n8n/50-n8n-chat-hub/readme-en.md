### 1ï¸âƒ£ Chat Hub Core Concepts

- Chat Hub is an integrated chat interface launched in n8n 2.1.0 Beta
- Design purpose: Build an LLM-agnostic moat where all workflows run on n8n
- Four key features: Multi-model support, native features, message-triggered workflows, two Agent modes

### 2ï¸âƒ£ Unified Chat Interface Features

- **Multi-model switching**: Same chat window can switch between OpenAI, Claude, Gemini, DeepSeek
- **Context preservation**: Remembers previous conversation content after switching models
- **Native tools**: Voice input, file upload, Jina AI Web Search / SERP API

### 3ï¸âƒ£ Personal Agent

- Similar to Custom GPT with preset System Prompt
- Example: Universal Traditional Chinese translator
- Supports voice, search, file upload and other Chat Hub native features

### 4ï¸âƒ£ Workflow Agent

- Convert existing n8n workflows directly into Chat Hub agents
- Key settings:
  - Check `Make Chat Available in n8n Chat` in Chat Trigger
  - Upgrade AI Agent to version 3.1+
  - Enable `Enable Streaming` option
  - Press Publish to deploy workflow

### 5ï¸âƒ£ Enterprise Value & Limitations

**Four Key Values:**
1. Integrate tools and reduce costs
2. Simplify automation workflow usage
3. Empower developers
4. Centralized management and governance

**Current Limitations:**
- Web search only has two options (Jina AI / SERP API)
- No grouping feature (unlike GPT's Projects)
- Workflow Agent recommended with Streaming support
