## 【 3 大影片重點 】

* 1️⃣ **認識與入門 RAG 知識庫**：實作簡單的 RAG （Retrieval-Augmented Generation），進一步理解核心原理。
* 2️⃣ **n8n RAG Agent 現場實作教學**：操作發票上傳與金額查詢，從資料「嵌入」處理到 RAG 檢索應用。
* 3️⃣ **Google NotebookLM 特色應用解析**：RAG 最簡單的選擇， NotebookLM 建立你的知識庫，實測發票文件上傳與 RAG 檢索。

## 🎯 目標

* 🚀 學會 RAG AI Agent 知識庫的基本運作，掌握 RAG 技術應用。
* 🤖 實際操作 n8n RAG Agent，自動查詢與回答發票金額，從最簡單的情境搞懂。
* 📝 利用 Google NotebookLM 結合手動與自動化需求，靈活管理個人與團隊知識。

---

## 🛠️ 對應情境

* 想要用低門檻方式建立屬於自己的 AI 知識庫。
* 有大量 PDF、郵件、文檔需要自動摘要、關鍵問答與搜尋。
* 希望將 n8n 流程與 Google NotebookLM 筆記型 AI 結合，提高日常資訊整理和查找效率。

## 🎥 你可以從影片中獲得

* **n8n 實機示範解析**：從發票 PDF 上傳、文本嵌入、向量化到檢索查詢全程解析，體驗從問題到答案的每一步。
* **核心實作流程復盤**：看會 AI Agent 架構，資料嵌入、查詢、回傳再與 LLM 整合的完整知識技能
